# Ken 老師 Excel → 題庫 JSON 轉換器

> ⚠️ **此工具僅適用 Ken 老師提供的 Excel 格式**：欄位順序為「題目 / 選項一 / 選項二 / 選項三 / 選項四 / 正確答案 / 正確答案解析及錯誤回饋」。其他老師若格式不同，請另建對應的轉檔工具或先手動調整欄位。

把符合格式的 Excel 題庫（例如「初級科目一.xlsx」「中級科目一.xlsx」）一次轉成 `import.html` 可匯入的 JSON。

## 使用流程

1. 把 `.xlsx` 題庫檔案放進 `input/` 資料夾（可以一次放多個）
2. 在 Finder 裡**雙擊** `convert.command`
3. 終端機會逐檔詢問缺失的欄位（科目、章節、難度、題型風格…）
4. 轉好的 JSON 檔會輸出到 `output/` 資料夾，檔名含時間戳
5. **成功轉檔的原始 Excel 會自動從 `input/` 搬到 `output/`**（保留做歷史紀錄；若搬移時 output 已有同名檔，會加上時間戳避免覆蓋）
6. 在 app 的「匯入題目」頁面把 JSON 內容貼上即可

> 若轉檔失敗（例如欄位數不足），該 xlsx **會留在 `input/`** 方便你修正後重跑。

## Excel 必要格式

| 欄位順序 | 內容 |
|----------|------|
| 第 1 欄 | 題目 |
| 第 2 欄 | 選項一 |
| 第 3 欄 | 選項二 |
| 第 4 欄 | 選項三 |
| 第 5 欄 | 選項四 |
| 第 6 欄 | 正確答案（填寫完整答案文字，不是 A/B/C/D） |
| 第 7 欄 | 正確答案解析及錯誤回饋（選填） |

- 第 1 列為標題列，腳本會自動略過
- 全空列會自動略過
- 每個 Excel 會產生一個對應的 JSON 檔

## 轉換時會問你什麼

每個檔案開始轉換前，會問一輪「該檔所有題目共用的欄位」：

- `subject_no` / `subject` — 科目編號與名稱
- `chapter_no` / `chapter` — 章節編號與名稱
- `question_type` — 題型（`single` / `choice`，預設 `single`）
- `difficulty` — 難度（`easy` / `medium` / `hard`，會依檔名「初級/中級/高級」自動推測預設值）
- `question_style` — 題型風格（`情境題` / `反面題` / `比較題` / `綜合題`，可留空）

`question_no` 會依 Excel 順序自動從 1 編號，不需要手動輸入。

## 第一次執行

腳本會自動偵測並安裝 `openpyxl`（只會在第一次）。如果執行時 macOS 說「無法打開」，請到：

- 系統偏好設定 → 安全性與隱私權 → 允許 `convert.command` 執行

或在終端機手動跑：

```bash
python3 convert.py
```
