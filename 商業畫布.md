# 商業模式畫布（Business Model Canvas）

## 產品：絹的考試系統 — AI 智慧考試學習平台
## 目標客群：傳統補習班 / 證照培訓機構
## 定價：NT$100,000 / 套

---

## 1. 目標客群（Customer Segments）

| 客群 | 描述 |
|------|------|
| **證照補習班** | TOEIC、金融證照、不動產經紀人、勞安衛等各類證照考試補習班 |
| **職業訓練機構** | 政府委辦職訓中心、企業內訓部門 |
| **專業培訓單位** | 會計師、勞資事務師等專業領域培訓班 |
| **中小型補習班** | 5-50 人規模，缺乏自建系統能力，但想數位轉型 |

**客群痛點：**
- 出題改考卷耗時耗力，老師時間成本高
- 無法追蹤每位學員的學習弱點
- 學員缺乏動機自主複習
- 傳統紙本考試無數據分析能力
- 想數位化但沒有 IT 人員、預算有限

---

## 2. 價值主張（Value Propositions）

### 核心價值：「讓補習班用 10 萬就擁有百萬級的 AI 學習平台」

| 價值 | 說明 |
|------|------|
| **AI 弱點分析** | GPT-4o 自動分析每位學員弱科，產出個人化學習建議報告 |
| **遊戲化學習** | 等級制度、經驗值、暴擊倍率、排行榜，讓學員主動刷題 |
| **白牌客製品牌** | 補習班的 Logo、名稱、色系，學員看到的是「自家的系統」 |
| **零技術門檻** | JSON 批次匯入題庫，不需要 IT 人員就能上線 |
| **行動裝置優先** | PWA 支援離線使用，手機隨時隨地練題 |
| **精細權限管理** | 依班級/標籤開放不同章節，區隔不同期別學員 |

**一句話賣點：**
> 「你的學員會主動打開手機刷題，而不是被逼著來補習班考試。」

## 3. 通路（Channels）

| 階段 | 通路 | 做法 |
|------|------|------|
| **認知** | Facebook/IG 廣告 | 投放補教業社群，強調「學員主動刷題」的痛點解決 |
| **認知** | 教育展 / 補教聯合會 | 現場 Demo，讓業者直接體驗 AI 分析效果 |
| **評估** | 免費試用帳號 | 提供 7 天試用，預載範例題庫 |
| **購買** | 直銷 / 業務拜訪 | 針對中大型補習班主動拜訪 |
| **交付** | GitHub Pages + Supabase | 線上開通，1-3 天完成部署 |
| **售後** | LINE 群組支援 | 技術問題即時解答 |

---

## 4. 客戶關係（Customer Relationships）

| 類型 | 內容 |
|------|------|
| **專屬客服** | 每位客戶一個 LINE 群組，含技術支援 |
| **初期導入協助** | 免費協助第一次題庫匯入與系統設定（含 organization_settings 品牌設定） |
| **教育訓練** | 提供管理後台操作教學（admin 標籤管理、章節權限、題庫管理） |
| **定期更新** | 功能更新自動同步至客戶部署分支 |

---

## 5. 收益流（Revenue Streams）

| 項目 | 金額 | 說明 |
|------|------|------|
| **系統導入費** | NT$100,000（一次性） | 含品牌設定、Supabase 專案建立、部署上線 |
| **年度維護費**（建議） | NT$24,000/年 | 系統更新、Bug 修復、LINE 技術支援 |
| **題庫代建服務**（加購） | NT$5,000-20,000/次 | 協助將紙本考題數位化、匯入系統 |
| **AI 分析用量**（加購） | 實報實銷 | OpenAI API 費用，約 NT$0.5/次分析 |
| **客製功能開發**（加購） | 依報價 | 特殊需求開發 |

**營收模型（保守估計）：**
- 首年取得 10 個客戶 → 導入費 NT$1,000,000
- 維護費 → NT$240,000/年（持續性收入）
- 題庫代建 → NT$100,000（額外收入）

## 9. 成本結構（Cost Structure）

| 項目 | 月成本 | 說明 |
|------|--------|------|
| **Supabase** | ~NT$0-800/客戶 | Free tier 可用，Pro plan $25 USD/月 |
| **GitHub Pages** | NT$0 | 免費靜態網站托管 |
| **OpenAI API** | ~NT$500-2,000/客戶 | 依 AI 分析使用量，GPT-4o-mini 成本極低 |
| **人力（開發）** | 依規模 | 目前一人可維運 |
| **人力（業務）** | 依規模 | 初期可兼任 |
| **行銷費用** | NT$5,000-20,000/月 | 社群廣告、教育展攤位 |

**關鍵數字：每個客戶的邊際成本約 NT$1,000-3,000/月，毛利極高。**

## 競爭優勢總結

| vs 競品 | 我們的優勢 |
|---------|-----------|
| **vs 自建系統**（百萬級） | 10 萬即可擁有，功能不輸自建 |

| **vs 線上題庫 App** | 白牌品牌、客製化、學員管理權限 |

## 核心商業邏輯

用極低的邊際成本（Supabase + GitHub Pages）服務每個客戶，10 萬的定價對補習班來說是「便宜到不可思議」的數位轉型方案，同時年度維護費創造持續性收入。現有 3 個客戶案例就是最好的銷售武器。
