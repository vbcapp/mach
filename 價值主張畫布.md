# 價值主張畫布（Value Proposition Canvas）

## 產品：絹的考試系統 — AI 智慧考試學習平台
## 目標客群：傳統補習班 / 證照培訓機構
## 定價：NT$100,000 / 套

---

## 一、客群側（Customer Profile）

### 1. 客戶任務（Customer Jobs）

客戶在日常營運中需要完成的工作：

| 類型 | 任務 | 重要性 |
|------|------|--------|
| **功能性任務** | 出考卷測驗學員學習成效 | ⭐⭐⭐⭐⭐ |
| **功能性任務** | 批改考卷、統計成績 | ⭐⭐⭐⭐⭐ |
| **功能性任務** | 追蹤每位學員的學習進度與弱點 | ⭐⭐⭐⭐ |
| **功能性任務** | 管理不同班級/期別的學員與教材權限 | ⭐⭐⭐⭐ |
| **功能性任務** | 提供學員課後複習的管道 | ⭐⭐⭐ |
| **社會性任務** | 展現補習班的專業形象與數位化程度 | ⭐⭐⭐⭐ |
| **社會性任務** | 讓家長/學員感受到「這間補習班很用心」 | ⭐⭐⭐⭐ |
| **情感性任務** | 相信自己的教學方法有數據支撐 | ⭐⭐⭐ |
| **情感性任務** | 減少行政瑣事的焦慮，專注教學本身 | ⭐⭐⭐⭐ |

---

### 2. 痛點（Pains）

客戶在執行任務過程中遇到的阻礙與困擾：

| 嚴重度 | 痛點 | 說明 |
|--------|------|------|
| 🔴 極高 | **出題改卷耗時耗力** | 老師花大量時間在出題與批改，壓縮了備課與教學時間 |
| 🔴 極高 | **無法追蹤學員個別弱點** | 紙本考試只有分數，無法知道每個人錯在哪些知識點 |
| 🟠 高 | **學員缺乏自主複習動機** | 學員被動等待上課，課後不主動練習 |
| 🟠 高 | **紙本考試無數據分析能力** | 考完就結束，沒有歷史趨勢、沒有弱科分析 |
| 🟠 高 | **想數位化但沒有 IT 人員** | 知道要轉型，但沒有技術能力，也沒預算請工程師 |
| 🟡 中 | **學員管理零散** | 用 Excel、紙本、LINE 群組分散管理，效率低 |
| 🟡 中 | **無法差異化教學** | 全班用同一套考卷，無法針對個人弱點強化 |
| 🟡 中 | **數位化成本高昂** | 市面系統動輒百萬，小型補習班負擔不起 |

---

### 3. 獲益（Gains）

客戶期望的成果與好處：

| 期望度 | 獲益 | 說明 |
|--------|------|------|
| 🟢 必要 | **省時省力** | 出題、改卷、成績統計全自動化 |
| 🟢 必要 | **學員成績提升** | 有數據支撐的教學調整，提升考取率 |
| 🔵 期望 | **學員主動學習** | 學員願意自發打開手機練題 |
| 🔵 期望 | **數據化教學管理** | 一眼看出全班弱點分佈、個人學習曲線 |
| 🔵 期望 | **品牌專業形象** | 擁有自己品牌的學習系統，提升補習班形象 |
| 🟣 驚喜 | **AI 個人化學習建議** | 系統自動幫每位學員產出專屬弱點分析報告 |
| 🟣 驚喜 | **零技術負擔** | 不需要懂技術就能完整營運系統 |
| 🟣 驚喜 | **行動學習體驗** | 學員手機隨時隨地都能練題，不受時間地點限制 |

## 二、價值側（Value Map）

### 1. 產品與服務（Products & Services）

我們提供的具體產品與功能：

| 類別 | 產品/服務 | 對應客戶任務 |
|------|----------|-------------|
| **核心產品** | AI 智慧考試學習平台 | 出題、測驗、成績管理全覆蓋 |
| **核心功能** | AI 弱點分析引擎（GPT-4o） | 追蹤學員弱點、產出個人化報告 |
| **核心功能** | 遊戲化學習系統（等級/經驗值/排行榜） | 激發學員自主學習動機 |
| **核心功能** | 白牌品牌客製化 | 補習班 Logo、名稱、色系完整客製 |
| **核心功能** | 精細權限管理（標籤/章節） | 管理不同班級與期別學員 |
| **交付服務** | 系統導入與部署（1-3 天上線） | 降低導入門檻 |
| **支援服務** | LINE 群組專屬客服 | 即時技術問題解答 |
| **加購服務** | 題庫代建（紙本數位化） | 解決題庫數位化的門檻 |

---

### 2. 痛點解方（Pain Relievers）

我們如何解決客戶的每一個痛點：

| 客戶痛點 | 解方 | 具體做法 |
|----------|------|----------|
| 出題改卷耗時耗力 | **JSON 批次匯入 + 自動批改** | 一次匯入整套題庫，系統自動計分，老師完全不需手動改卷 |
| 無法追蹤學員弱點 | **AI 弱點分析報告** | GPT-4o 自動分析每位學員的錯題分佈，產出章節級弱點報告與學習建議 |
| 學員缺乏自主複習動機 | **遊戲化激勵機制** | 等級制度、經驗值、暴擊倍率、排行榜，把刷題變成像打電動一樣有成就感 |
| 紙本考試無數據分析 | **完整數據儀表板** | 正確率趨勢、章節弱點分佈、答題時間分析，全部視覺化呈現 |
| 想數位化但沒 IT 人員 | **零技術門檻設計** | 管理後台圖形化操作，題庫用 JSON 匯入，不需寫程式、不需架伺服器 |
| 學員管理零散 | **統一管理平台** | 學員帳號、成績、進度、權限全在同一個系統管理 |
| 無法差異化教學 | **個人化學習路徑** | AI 根據弱點自動推薦練習內容，每個學員看到不同的建議 |
| 數位化成本高昂 | **NT$100,000 一次買斷** | 只要 10 萬就擁有百萬級功能的系統，邊際成本極低 |

### 3. 獲益引擎（Gain Creators）

我們如何創造客戶期望（甚至超越期望）的價值：

| 客戶獲益 | 引擎 | 創造方式 |
|----------|------|----------|
| 省時省力 | **全自動化流程** | 從出題→測驗→批改→成績統計→弱點分析，全程零人工介入 |
| 學員成績提升 | **數據驅動教學** | 老師能根據 AI 分析報告調整教學重點，精準補強全班共同弱點 |
| 學員主動學習 | **遊戲化正循環** | 「練題→得經驗值→升等→排行榜競爭→再練題」的成癮循環 |
| 數據化教學管理 | **多維度分析報表** | 班級整體弱點分佈、個人學習曲線、章節正確率熱力圖 |
| 品牌專業形象 | **白牌客製品牌** | 學員看到的完全是補習班自己的品牌，提升專業感和信賴感 |
| AI 個人化建議 | **GPT-4o 深度分析** | 不只告訴學員「哪裡弱」，還給出「怎麼加強」的具體學習建議 |
| 零技術負擔 | **全託管式部署** | 我們負責建立、部署、維護，客戶只需專注在教學與管理 |
| 行動學習體驗 | **PWA 行動優先設計** | 支援離線使用，手機瀏覽器打開即用，不需下載 App |

## 三、適配分析（Fit Analysis）

### 問題—解方適配（Problem-Solution Fit）

| 配適度 | 評估 |
|--------|------|
| 痛點覆蓋率 | **8/8（100%）** — 所有已識別痛點皆有對應解方 |
| 獲益覆蓋率 | **8/8（100%）** — 所有期望獲益皆有對應引擎 |
| 核心痛點解決強度 | **極高** — 前三大痛點（出題改卷/弱點追蹤/學員動機）均有強力解方 |

### 差異化優勢

| 維度 | 我們 | 傳統替代方案 |
|------|------|-------------|
| 成本 | NT$100,000 一次性 | 自建系統百萬起跳 |
| 部署速度 | 1-3 天上線 | 自建 3-6 個月 |
| AI 分析 | GPT-4o 自動弱點分析 | 無，或需額外購買 |
| 學員動機 | 遊戲化系統驅動自主學習 | 傳統考試被動練習 |
| 品牌客製 | 完整白牌，學員感知為自家系統 | 通用品牌，無歸屬感 |
| 技術門檻 | 零門檻，無需 IT 人員 | 需要工程師維運 |

---

### 核心價值公式

> **以 1/10 的價格，提供 10 倍的效率提升，同時讓學員從「被迫考試」變成「主動刷題」。**
