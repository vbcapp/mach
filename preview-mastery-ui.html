🎯 UX 專家推薦

最佳方案：方案 4（首次提示 + 資訊按鈕）
理由：
  • 新用戶：首次看到黃色提示框，清楚知道規則
  • 老用戶：關閉提示後，介面保持簡潔
  • 忘記規則：隨時可點擊 ? 查看詳細說明
  • 跨平台：手機、桌面都好操作
實作建議：
1. 使用 localStorage 記錄是否已顯示過提示
2. 首次使用顯示黃色提示框（含動畫吸引注意）
3. 用戶點擊「我知道了」後，設定 localStorage，隱藏提示
4. 保留資訊按鈕，點擊彈出詳細說明彈窗

📚 什麼是「精熟」？

🎯
精熟定義
當你將某一題連續答對 3 次，就算精熟這題！
💡
為什麼是 3 次？
這代表你真的理解了，不是靠猜測或短期記憶
📉
精熟後的好處
精熟的題目會減少出現頻率，讓你專注在還不熟的題目
