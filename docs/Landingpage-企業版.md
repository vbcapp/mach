### 1. Hero Section

**適用定律：** 希克定律（1 個 CTA）、費茲定律（大按鈕）、系列位置效應（首尾最重要）

| 元素 | 內容 |
|------|------|
| 主標題 | **企業內訓，終於能「量化成效」了** |
| 副標題 | 一站搞定出題、測驗、成果追蹤，讓每一筆培訓預算都花在刀口上 |
| CTA 按鈕 | **預約免費 Demo** |
| 輔助文字 | ✦ 15 分鐘線上演示 ✦ 不需綁約 |
| 視覺 | 管理者後台儀表板截圖（顯示圖表與數據） |

**設計要點：**
- CTA 按鈕用 `#FFD600` 金色底 + 4px 黑框 neo-brutalism 風格
- B2B 的 CTA 是「預約 Demo」而非「立即註冊」（決策鏈更長，先降低門檻）
- 主標直擊 HR 痛點：「量化成效」是企業培訓最大的缺口
- 按鈕尺寸至少 48px 高、寬度 240px+
