## Slide 2 — 我們的做法

先刪。

| 傳統 LMS 有的 | 我們刪掉了 |
|-------------|-----------|
| 影片托管 | 刪。PDF 夠了 |
| 複雜權限系統 | 刪。標籤就夠 |
| 客製 API | 刪。沒人用 |
| 50 頁管理後台 | 刪。剩 3 頁 |
| 心理測評 | 刪。才庫旗艦版才加回來 |

剩下的三件事：

1. **丟 PDF，AI 5 分鐘建好課** — 省講師 3 天
2. **翻卡 + 升等 + 排行榜** — 員工自己打開
3. **課後 3/7/14 天追落實** — 老闆第一次看到行為改變

**NT$100,000/年。**
原材料 5 萬，我們收 10 萬。白痴指數 2。
市場是 10。

## Slide 3 — 試行

**驗證一件事：有人願意付錢嗎？**

免費試用不算數。免費試用只證明有人願意「用」，不證明有人願意「付」。

**規則：**
- 3 個月
- 2 家企業（製造業或服務連鎖，不碰金融業）
- 試行費 NT$50,000，可折抵年費
- 從才庫現有客戶裡找最熟的一家先開始

**3 個月後只回答三件事：**

1. 有人付錢？ → 推
2. 沒人付錢？ → LMS 打包進培訓服務，不單獨賣
3. 價格帶在哪裡？ → 填三個空格：

| 太便宜 | 合理 | 太貴 |
|-------|-----|-----|
| <\_\_\_ | \_\_\_ 至 \_\_\_ | >\_\_\_ |

---

**成本：** 10 小時人力/家 + 5,000 API 費。
**失敗代價：** 2 萬台幣 + 3 個月。
**成功收益：** 驗證一套可複製的定價模型。

沒有第四頁。
