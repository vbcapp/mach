## 二、各區塊文字內容

---

### 1. Hero Section

**適用定律：** 希克定律（1 個 CTA）、費茲定律（大按鈕）、系列位置效應（首尾最重要）

| 元素 | 內容 |
|------|------|
| 主標題 | **刷題備考不再盲目** |
| 副標題 | AI 幫你抓弱點，省下 70% 無效練習 |
| CTA 按鈕 | **免費開始刷題** |
| 輔助文字 | ✦ 不需信用卡 ✦ 30 秒註冊 |
| 視覺 | 手機 mockup 展示「弱點分析」或「章節雷達圖」畫面 |

**設計要點：**
- CTA 按鈕用 `#FFD600` 金色底 + 4px 黑框 neo-brutalism 風格
- 按鈕尺寸至少 48px 高、寬度 240px+（費茲定律：大目標易點擊）
- Hero 區只有 1 個行動選項（希克定律：選項越少決策越快）
- 主標放頁面最頂部（系列位置：首位效應）

**適用定律：** 格式塔（接近性 + 相似性分組）、米勒定律（4-5 組 ≤ 7±2）

| 元素 | 內容 |
|------|------|
| 區塊標題 | **為什麼聰明人都用考試神器？** |
| 功能卡 1 | 🎯 **AI 弱點狙擊** — 系統自動分析你最容易錯的題目，精準推薦複習順序，不再浪費時間重複刷已經會的題。 |
| 功能卡 2 | ⚡ **即時作答反饋** — 每答一題馬上知道對錯與詳細解析，不用等到全部考完才知道結果。零碎時間也能有效學習，通勤、午休隨手刷幾題就有收穫。 |
| 功能卡 3 | 📊 **即時戰力儀表板** — 正確率趨勢、章節雷達圖、考試倒數，一眼掌握你的備考進度和盲區。 |
| 功能卡 4 | 🏆 **排行榜 ＋ 等級系統** — XP 升等、暴擊加倍、每日排名，把枯燥備考變成破關遊戲。 |

**設計要點：**
- 前 4 張卡片用 2×2 grid（手機端：1 欄堆疊）
- 每張卡片結構統一：icon → 標題 → 1-2 句說明（格式塔相似性）
- 卡片間距一致 16px（格式塔接近性）
- 控制在 4-5 項（米勒定律：認知負荷 ≤ 7±2）
- 第 5 張可做滿寬橫幅，視覺節奏打破單調
